Импорт товаров из CSV
Выберите CSV-файл для загрузки. Система обновит существующие товары (по `id`) или создаст новые, если `id` не указан или не найден.
Формат CSV файла

Для правильного импорта товаров, CSV файл должен содержать следующие колонки:

Поле Тип Обязательно Описание
id Integer Нет Уникальный идентификатор товара. Если не указан, будет создан новый товар.
name String Да Название товара
slug String Нет URL-идентификатор товара (генерируется автоматически если не указан)
description Text Нет Описание товара
price Decimal Да Цена товара
old_price Decimal Нет Старая цена (для отображения скидки)
sku String Нет Артикул товара
in_stock Integer Да Количество на складе
category_name String Нет Название категории (будет создана если не существует)
brand_name String Нет Название бренда (будет создан если не существует)
meta_title String Нет SEO заголовок страницы
meta_description Text Нет SEO описание страницы
